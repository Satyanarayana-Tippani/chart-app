softwares required
=================
1) Node js

React plugins used
==================
1) react-checkbox-tree  (https://www.npmjs.com/package/react-checkbox-tree)
2) react-google-charts  (https://www.npmjs.com/package/react-google-charts)
3) drawer UI - (https://material-ui.com/components/drawers/)

Steps:
========
1) to this current path (Chart_React_App) and open terminal and execute command "npm install" will install node modules
2) execute command "npm start"
3) open browser to launch app "http://localhost:3000/"
